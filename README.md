# geometry_lib

Библиотека для расчёта площади и периметра фигур.

## Функции

### rectangle.py
`area(a, b)` — площадь прямоугольника  
Пример: `area(5, 3)` → 15

`perimeter(a, b)` — периметр прямоугольника  
Пример: `perimeter(5, 3)` → 16

### circle.py
`area(r)` — площадь круга  
Пример: `area(3)` → 28.274333882308138

`perimeter(r)` — длина окружности  
Пример: `perimeter(3)` → 18.84955592153876

### square.py
`area(a)` — площадь квадрата  
Пример: `area(4)` → 16

`perimeter(a)` — периметр квадрата  
Пример: `perimeter(4)` → 16

### triangle.py
`area(a, h)` — площадь треугольника  
Пример: `area(6, 4)` → 12.0

`perimeter(a, b, c)` — периметр треугольника  
Пример: `perimeter(3, 4, 5)` → 12
